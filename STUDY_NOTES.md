## 0. 큰 그림 — 한 문장 요약

CVD 증착률은 **저온에서는 표면 반응 속도**가, **고온에서는 반응물 공급(확산) 속도**가 발목을 잡으며, 이 두 단계가 직렬로 일어나기 때문에 **항상 느린 쪽이 전체 속도를 결정**한다. Arrhenius 분석으로 저온 영역에서 활성화에너지 `Ea`를 뽑아낼 수 있다.

---

## 1. CVD란 무엇인가 (배경)

**CVD (Chemical Vapor Deposition, 화학기상증착)**: 기체 상태의 전구체(precursor)를 챔버에 넣고, 가열된 기판 표면에서 화학반응을 일으켜 고체 박막을 쌓는 공정.

반도체에서의 쓰임 (확인된 사실):
- **SiO₂** 절연막 — 전구체 예: SiH₄ + O₂, 또는 TEOS
- **poly-Si** (다결정 실리콘) 게이트 — 전구체 예: SiH₄ 열분해
- **Si₃N₄** (질화막) — 전구체 예: SiH₄ + NH₃
- **W, TiN** 등 금속/배리어 — 배선 및 확산방지막

비교 개념: **PVD (Physical Vapor Deposition)** 는 화학반응 없이 물리적으로(증발·스퍼터링) 막을 올린다. 이 프로젝트는 CVD에 초점.

> **추측입니다**: 학연생 컨택에서 "왜 CVD를 골랐나" 질문이 나온다면, "공정 변수(온도·압력·분압)와 증착률의 정량 관계를 모델링하기 좋아서"라고 답하면 자연스럽다.

## 2. 핵심 개념 — 율속 단계 (Rate-Limiting Step)

증착은 최소 두 단계가 **직렬**로 일어난다.

```
   기상의 반응물  ──①공급(확산)──▶  표면  ──②화학반응──▶  고체 박막
```

직렬 과정에서는 **가장 느린 단계가 전체 속도를 결정**한다.
비유: 좁은 관 두 개를 물이 차례로 통과하면, 더 좁은 관이 전체 유량을 정한다.

| 영역 | 율속 단계 | 온도 올리면? | 지배 온도 |
|---|---|---|---|
| 반응 율속 (reaction-limited) | 표면 화학반응 | 지수적으로 빨라짐 | 저온 |
| 물질전달 율속 (mass-transport-limited) | 기상 확산 공급 | 거의 안 변함 | 고온 |

**왜 이렇게 갈리나** (확인된 사실): 화학반응 속도는 온도에 지수적으로 민감(Arrhenius)하지만, 기상 확산은 온도에 약하게(거듭제곱) 의존한다. 그래서 온도를 올리면 반응은 폭발적으로 빨라져 더 이상 병목이 아니게 되고, 상대적으로 느린 확산이 병목이 된다.

---

## 3. 공식 ① — Arrhenius 식 (저온, 반응 율속)

$$R = A \cdot \exp\!\left(-\frac{E_a}{k_B T}\right)$$

| 기호 | 의미 | 코드 변수 |
|---|---|---|
| `R` | 증착률 [nm/min] | `R_rxn` |
| `A` | 전지수 인자 (충돌 빈도 등 상수) | `A_true = 5.0e7` |
| `Ea` | 활성화에너지 (넘어야 할 에너지 장벽) [eV] | `Ea_true = 0.85` |
| `k_B` | 볼츠만 상수 | `k_B_eV = 8.617e-5 eV/K` |
| `T` | 절대온도 [K] | `T` |

**활성화에너지 `Ea`의 직관**: 반응이 일어나려면 반응물이 일정 에너지 장벽을 넘어야 한다. `Ea`가 클수록 온도에 더 민감하게 반응속도가 변한다.

**왜 eV를 쓰나** (확인된 사실): `k_B`를 eV/K 단위로 쓰면 `Ea`가 곧장 eV로 나온다. CVD 활성화에너지는 보통 0.5~2 eV 범위 (코드 주석 근거).

### 로그 변환 — 이게 분석의 핵심

양변에 자연로그:

$$\ln R = \ln A - \frac{E_a}{k_B}\cdot\frac{1}{T}$$

이건 `y = b + m·x` 꼴의 **직선**이다.
- x축 = `1/T`
- y축 = `ln R`
- **기울기 `m = −Ea/k_B`**
- y절편 `b = ln A`

→ 따라서 **기울기만 재면 `Ea = −m · k_B`** 로 역산.

코드 대응:
```python
slope, intercept = np.polyfit(invT_fit, lnR_fit, 1)  # 직선 피팅
Ea_fit = -slope * k_B_eV                              # 기울기 → Ea
A_fit  = np.exp(intercept)                            # 절편 → A
```

## 4. 공식 ② — 물질전달 (고온)

$$R \propto T^{n}, \quad n \approx 1.5 \sim 2$$

코드:
```python
def rate_mass_transport(T, scale, n):
    return scale * (T / 1000.0) ** n   # n_mtl = 1.7
```

**왜 거듭제곱인가** (확인된 사실 + 보충): 기상 확산계수 `D`가 온도에 `D ∝ T^(1.5~1.8)` 정도로 의존하기 때문. 지수함수(Arrhenius)와 비교하면 훨씬 완만 → 온도를 올려도 증착률이 거의 안 변함.

> **확실하지 않음**: `n ≈ 1.5~2`의 정확한 값은 전구체·압력·반응기 형상에 따라 달라진다. 코드/README에 출처가 명시돼 있지 않으므로, 실제 논문 데이터로 확장할 때 문헌값으로 확인할 것.

## 5. 공식 ③ — 두 영역의 합성 (이 코드의 백미)

$$\frac{1}{R_{total}} = \frac{1}{R_{reaction}} + \frac{1}{R_{transport}}$$

코드:
```python
R_total_clean = 1.0 / (1.0 / R_rxn + 1.0 / R_mtl)
```

**전기회로 비유** (확인된 사실): 직렬 저항의 합과 같은 구조. 두 "저항"(=1/속도) 중 **큰 쪽**(=느린 단계)이 합을 지배한다.
- 저온: `R_rxn`이 작음(반응 느림) → `1/R_rxn`이 큼 → 반응이 율속
- 고온: `R_mtl`이 작음(확산 느림) → `1/R_mtl`이 큼 → 확산이 율속

그래프 (c)에서 주황선·초록선의 **교차점**이 영역 전환점이고, 관측되는 실제 증착률(검은 굵은 선)은 항상 두 선 중 **아래쪽(느린 쪽)**을 따라간다.

---

## 6. 공정 윈도우 (Process Window) — 왜 고온을 선호하나

코드:
```python
dlnR_dT = np.gradient(np.log(R_total_clean), T)   # 온도 민감도
stable = np.abs(sensitivity_pct) < 0.3            # < 0.3 %/K 영역
```

**온도 민감도** `d(ln R)/dT`: 온도가 1 K 흔들릴 때 증착률이 몇 % 변하는지.

**핵심 통찰** (확인된 사실): 물질전달 율속(고온) 영역은 온도 의존성이 약하다 = 온도가 조금 흔들려도 증착률이 거의 안 변한다 = **재현성(reproducibility)이 우수**. 그래서 실제 양산 공정에서는 일부러 이 영역을 선호하는 경우가 많다.

> **추측입니다**: 이건 "감도가 낮은 게 오히려 좋다"는 공정 엔지니어 특유의 사고방식을 보여주는 대목이라, 컨택에서 언급하면 실무 감각이 있어 보인다.

코드 결과: 약 630~830 °C 구간이 민감도 < 0.3 %/K → 안정 윈도우.

## 7. 가장 중요한 학습 포인트 — 경계 설정이 Ea를 좌우한다

`ln(R) vs 1/T`가 직선인 건 **순수 반응 율속 영역에서만** 성립한다. 물질전달 영향이 섞인 점까지 직선 피팅에 넣으면 기울기가 완만해지고 **Ea가 과소평가**된다.

| 경계 설정 | 추출 Ea | R² | 사용 점 수 |
|---|---|---|---|
| T < 850 K (느슨) | 0.557 eV | 0.981 | 12 |
| T < 760 K (엄격) | 0.643 eV | 0.991 | 8 |
| (참값) | 0.85 eV | — | — |

**트레이드오프** (확인된 사실): 경계를 좁힐수록 참값에 가까워지지만, 점 개수가 줄어 통계가 불안정해진다. "어디까지가 순수 반응 율속 영역인가"를 판단하는 안목이 분석의 핵심.

> **추측입니다**: 면접·컨택에서 이 프로젝트를 한 문장으로 요약한다면 — "합성 데이터로 Arrhenius를 그렸다"가 아니라 **"영역 경계 설정이 Ea 추출에 주는 편향을 정량적으로 보였다"** 가 훨씬 강하다.

1. ① 기상 확산을 통한 반응물 공급, ② 표면 화학반응

## 9. 용어 빠른 사전

| 한글 | 영어 | 한 줄 정의 |
|---|---|---|
| 활성화에너지 | activation energy (Ea) | 반응이 일어나려 넘어야 할 에너지 장벽 |
| 전지수 인자 | pre-exponential factor (A) | Arrhenius 식의 상수, 충돌 빈도 등 |
| 율속 단계 | rate-limiting step | 직렬 과정에서 전체 속도를 정하는 가장 느린 단계 |
| 반응 율속 | reaction-limited | 표면 반응이 병목인 (저온) 영역 |
| 물질전달 율속 | mass-transport-limited | 반응물 공급(확산)이 병목인 (고온) 영역 |
| 공정 윈도우 | process window | 안정적으로 재현되는 공정 변수 범위 |
| 전구체 | precursor | 박막의 원료가 되는 기체 분자 |
